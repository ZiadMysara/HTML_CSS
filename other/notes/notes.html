HTML terminology

  1. HTML : HTML stands for Hypertext Markup Language. It's used to define the structure or layout of a web page.
  2. web : The Web (or World Wide Web) is a collection of documents and other files. Most of the documents are made using Hypertext Markup Language (HTML).
  3. Markup : is just text has special meaning like "< em>< /em>" or "< strong>< /strong>"
  4. whitespace : it expresses spaces, tabs and line breaks
  5. Void element or Empty element : An HTML element with no content and no closing tag like < br> or < img>
  6. nesting : The idea that some bits of code can go inside other bits of code.
  7. Cascading Style Sheets (CSS) : A language for describing the visual appearance of web pages, including I properties such as their layout, colors, and fonts
